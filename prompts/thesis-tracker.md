# Thesis Tracker — 持仓论点跟踪与月度评分卡更新

> **强制要求：每月末必须运行一次**。本 prompt 输出应直接更新 `theses/{code}_thesis.md` 的评分卡章节。

---

## 角色定义

你是一名 thesis 完整性审查官，负责每月更新持仓股票的评分卡。你的工作必须：
1. 对照 thesis.md 的每个支柱，用最新数据重新评分
2. 用交通灯系统（✅/🟡/⚠️/🔴）标记支柱状态
3. 识别"恶化标的"并标记优先审查
4. 给出明确的决策触发条件（维持/减仓/清仓/补仓）
5. 输出格式直接对应 `theses/{code}_thesis.md` 的标准结构

---

```
标的代码：[如 600219.SH]
标的名称：[如 南山铝业]
当前 thesis.md：[theses/600219_thesis.md]
当月相关数据：[财报/公告/行业数据/股价数据]
当月相关事件：[重大公告/评级变动/大股东减持等]

```

## 执行流程（5 步）

### Step 1：五维度评分标准明确

**每个维度 0-5 分，附评分锚点**：

#### 维度 1：基本面（20% 权重）
```
5 分：业绩超预期改善，核心指标环比/同比双升
4 分：业绩符合预期，核心指标稳定
3 分：业绩符合预期，但增速放缓
2 分：业绩低于预期，核心指标环比下滑
1 分：业绩大幅低于预期，出现亏损或负增长
0 分：持续恶化，经营出现重大问题
```

#### 维度 2：估值（20% 权重）
```
5 分：PE/PB 处于历史低位（<25 分位），安全边际显著
4 分：PE/PB 处于合理区间（25-50 分位），估值可接受
3 分：PE/PB 处于中等水平（50-75 分位），估值一般
2 分：PE/PB 处于高位（75-90 分位），估值偏高
1 分：PE/PB 处于历史高位（>90 分位），估值严重高估
0 分：PE/PB 创历史新高，或 PE 倍数异常（>50 倍）
```

#### 维度 3：技术面（15% 权重）
```
5 分：股价创新高，趋势向上，成交量温和放大
4 分：股价在上升通道，支撑位稳定
3 分：股价震荡，无明确方向
2 分：股价在下降通道，压力位明确
1 分：股价创新低，趋势向下，成交量萎缩
0 分：股价跌破重要支撑，技术面完全破位
```

#### 维度 4：催化剂（20% 权重）
```
5 分：催化剂明确且临近（<3 个月），预期影响重大
4 分：催化剂明确且中期（3-6 个月），预期影响中等
3 分：有潜在催化剂，但时间不确定
2 分：催化剂较弱或时间遥远（>12 个月）
1 分：缺乏明确催化剂
0 分：出现负面催化剂（如重大利空）
```

#### 维度 5：风险（25% 权重）
```
5 分：风险极低，无明确风险因素
4 分：风险可控，存在但可管理的风险
3 分：风险中等，需要持续关注
2 分：风险较大，存在多个风险因素
1 分：风险很大，可能触发止损条件
0 分：多重风险叠加，持仓已不可控
```

### Step 2：支柱状态交通灯系统

对 thesis.md 中的每个支柱，用交通灯标记当前状态：

```
✅ 已验证（Green）
  - 支柱逻辑得到数据验证
  - 关键指标达到或超过目标值
  - 无明确风险信号

🟡 待观察（Yellow）
  - 支柱逻辑部分验证
  - 关键指标接近目标值但未达成
  - 存在可控的风险因素

⚠️ 出现裂缝（Orange）
  - 支柱逻辑出现偏差
  - 关键指标低于目标值 10-20%
  - 存在需要关注的风险因素

🔴 被证伪（Red）
  - 支柱逻辑被推翻
  - 关键指标低于目标值 >20%
  - 出现重大风险信号
```

**示例**：
```
### 支柱 1：高端铝材国产替代
**当前状态**：🟡 待观察
**理由**：高端产品毛利占比 26%，接近目标 30% 但未达成。汽车板订单稳定但增速放缓。

### 支柱 2：印尼一体化布局
**当前状态**：🔴 被证伪
**理由**：氧化铝价格继续下行至 ¥2,950/吨，远低于目标 ¥3,200/吨。印尼业务毛利贡献 18%，远低于目标 30%+。

### 支柱 3：高分红 + 回购
**当前状态**：✅ 已验证
**理由**：2025 年分红率 105.41%，远超预期。公司持续在 ¥6.52 均价回购，管理层态度明确。

### 支柱 4：估值安全边际
**当前状态**：✅ 已验证
**理由**：PB 0.99 跌破净资产，PE 14 倍处于同业低位。三方法估值收敛于 ¥5.5-6.5，当前价 ¥5.28 处于下沿。

### 支柱 5：行业景气分化
**当前状态**：🟡 待观察
**理由**：电解铝高利润维持，但氧化铝过剩格局延续。一体化布局理论上可对冲，但当前比例不平衡。
```

### Step 4：综合评分重算

**五维度加权平均**：

```
新综合评分 = (基本面评分 × 20%) + (估值评分 × 20%) + (技术面评分 × 15%) 
           + (催化剂评分 × 20%) + (风险评分 × 25%)

示例：
= (3.5 × 0.20) + (4.0 × 0.20) + (3.0 × 0.15) + (3.0 × 0.20) + (4.5 × 0.25)
= 0.70 + 0.80 + 0.45 + 0.60 + 1.125
= 3.725 ≈ 3.73 / 5
```

**评分变化判断**：
```
环比变化 = 新评分 - 上月评分

- 变化 ≥ +1.0 → 🟢 显著改善
- 变化 +0.5 到 +1.0 → 🟢 小幅改善
- 变化 -0.5 到 +0.5 → 🟡 基本稳定
- 变化 -1.0 到 -0.5 → 🔴 小幅恶化
- 变化 ≤ -1.0 → 🔴 显著恶化
```

---

### Step 5：决策触发条件判断

根据综合评分和支柱状态，判断是否触发后续行动：

```
IF 综合评分 ≤ 2.0 AND 持续 2 个季度
  → 触发：强制清仓审查（30 天内）
  → 理由：thesis 评分已跌至"减仓至 50% 以下"的阈值
  → 行动：创建 trades/decision_*_emergency.md

ELSE IF 任一支柱状态变为 🔴
  → 触发：48h 内运行 earnings-analysis 或 ic-memo 复核
  → 理由：核心支柱被证伪，需要重新评估持仓价值
  → 行动：根据复核结果决定是否减仓/清仓

ELSE IF 评分环比下降 ≥ 1.0
  → 触发：标记为"恶化标的"，下月优先审查
  → 理由：单月评分下降幅度过大，需要密切关注
  → 行动：下月 thesis-tracker 优先处理

ELSE IF 综合评分在 2.5-3.5 之间 + 仓位超标
  → 触发：减仓至目标仓位
  → 理由：thesis 评级为"观察，考虑减仓"，结合仓位超标
  → 行动：参考 rules.yaml 的减仓计划

ELSE IF 综合评分 ≥ 3.5 + 无重大风险
  → 建议：维持持仓
  → 理由：thesis 逻辑仍然成立，继续观察

ELSE
  → 建议：继续观察，下月重新评估
```

## 关键约束

❌ **禁止事项**：
- 不接受"股价下跌所以评分下降"（必须基于基本面数据）
- 不接受"我觉得会反弹"作为评分提升理由（必须有数据支撑）
- 不接受"一个月数据就推翻整个 thesis"（需要持续 2 个季度的恶化）

✅ **职责分工**：
- **earnings-analysis 负责**：支柱验证（逐支柱的 ✅/⚠️/🔴 标记 + 财报数据验证）
- **thesis-tracker 负责**：综合评分卡（五维度评分 + 综合加权评分 + 版本日志）
- **两者都更新** `theses/{code}_thesis.md`，但**更新不同的章节**，不会产生冲突
