## 二、三情景目标价

### 悲观情景（30%）

**假设**：
- 芯片需求增速放缓至 5%
- 华天科技市场份额无提升，反而因竞争加剧下滑 5%
- PE 倍数回归行业均值 25 倍
- 净利润同比增长 -10%

**目标价计算**：
```
2025 年 EPS ≈ 710.51M / 市值 60.99B ≈ ¥0.27
悲观情景 EPS = ¥0.27 × 0.9 = ¥0.24
悲观情景目标价 = ¥0.24 × 25 倍 = ¥6.00
```

---

### 基础情景（50%）

```
基础情景 EPS = ¥0.27 × 1.15 = ¥0.31
基础情景目标价 = ¥0.31 × 30 倍 = ¥9.30
```

---

## 三、加权期望回报

```
加权期望目标价 = (¥6.00 × 0.3) + (¥9.30 × 0.5) + (¥11.90 × 0.2)
                = ¥1.80 + ¥4.65 + ¥2.38
                = ¥8.83

加权期望回报 = (¥8.83 / ¥20.54) - 1 = -57.0%
```

## 四、估值三角形验证

### 方法 A：PE 估值法

```
当前 PE：75.96 倍（极高）
行业均值 PE：25-35 倍
合理 PE：30 倍（保守估计）

基础情景 EPS：¥0.31
合理目标价 = ¥0.31 × 30 = ¥9.30
```

### 方法 B：PB 估值法

```
当前 PB：未知（需补充）
假设 PB ≈ 3 倍（行业均值）
每股净资产 ≈ ¥20.54 / 3 ≈ ¥6.85

合理目标价 = ¥6.85 × 3 = ¥20.54（当前价）
这意味着当前 PB 已经处于高位
```

### 方法 C：分红折现法

```
2025 年分红：¥0.02（极低，分红率 <1%）
假设分红增长率：5%
折现率：8%

目标价 = ¥0.02 × 1.05 / 0.08 = ¥0.26
```

**收敛判断**：
- ❌ 三方法结果严重分散：¥0.26 - ¥20.54
- ❌ 当前价格 ¥20.54 远高于 PE 法和分红法的合理价
- ❌ 估值严重不合理

## 五、风险回报比评估

```
最大亏损 = ¥20.54 - ¥6.00 = ¥14.54（-70.8%）
加权期望回报 = -57.0%
风险回报比 = (-57.0%) / (-70.8%) = 0.80

但这个比率掩盖了真实问题：
- 你的期望回报是负的（-57%）
- 最大亏损也是负的（-70.8%）
- 这不是"风险可控"，这是"亏损确定"
```

**判断标准**：
- ❌ 风险回报比 0.80 看似可控
- ❌ 但前提是你期望赚钱，而这里你期望亏钱
- ❌ **这个决策本质上是在赌股价继续上涨，而非基本面投资**
